# Nvidia (Telecom)

## Latest
- **Mar 3, 2026**: Deepened alliance with **Nokia** with a **$1 billion investment** from Nvidia to accelerate AI-native mobile network development.
- **Mar 2, 2026**: Launched the **Nemotron Large Telco Model (LTM)**, a 30-billion-parameter model developed with AdaptKey AI for autonomous network operations (fault detection, planning).
- **Mar 2, 2026**: Formed a global **AI-Native 6G Coalition** with leaders including Ericsson, Nokia, SoftBank, T-Mobile, and Deutsche Telekom.
- **Mar 2, 2026**: Demonstrated concurrent AI and RAN processing with **T-Mobile US** using Nokia's CUDA-accelerated software on Nvidia's Aerial RAN Computer (ARC).
- **Mar 2, 2026**: Announced **Ericsson Cloud RAN** software is now portable to Nvidia AI infrastructure via Nvidia Aerial CUDA.
- **Feb 2026**: **SoftBank** achieved industry-first 16-layer massive MIMO using fully software-defined 5G on Nvidia hardware.

## Profile
- **Role**: Catalyst for the "AI-RAN" movement, providing GPU-accelerated software-defined infrastructure for telecom.
- **Key Focus**: AI-Native 6G, GPU-based RAN (Aerial RAN Computer), and Large Telco Models (LTM).
- **Strategic Pivot**: Moving from being a "chip provider" to a core "telecom infrastructure partner" through massive R&D investments (e.g., Nokia) and ecosystem building (AI-RAN Alliance).
