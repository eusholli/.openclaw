# Iridium Communications

## Latest
- **Project Stardust (2026 Launch):** A major pivot to standards-based (3GPP Rel-17+) Non-Terrestrial Network (NTN) service. It enables narrowband IoT (NB-IoT) and direct-to-device (D2D) messaging using their existing 66 LEO satellite constellation with software upgrades.
- **Customer Management Platform (CMP) RFx:** Iridium is currently evaluating a new CMP to handle the massive scale of subscribers expected from the 3GPP pivot (shifting from niche specialized hardware to mass-market 5G devices).
- **Deutsche Telekom Partnership (2025):** DT selected Iridium for its NB-IoT D2D roaming service, providing global connectivity for IoT and automotive use cases starting in 2026.

## Profile
- **Role:** Global leader in Low Earth Orbit (LEO) satellite communications.
- **Strategic Focus:** Non-terrestrial networks (NTN), Direct-to-Device (D2D), and "standardized" satellite-to-cell services.
- **Headquarters:** McLean, Virginia (DC area).
